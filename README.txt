# Turing Machine Simulator

This project is a graphical Turing Machine Simulator built using Python and Tkinter.

## Features

- Load a Turing Machine from a text file
- Simulate the input string on the tape
- Show the tape as cells
- Animate the tape head movement step by step
- Display Accepted or Rejected result
- No external libraries required

## How to Run

1. Open the project folder in Visual Studio Code.
2. Make sure Python 3 is installed.
3. Run:

```bash
python main.py
